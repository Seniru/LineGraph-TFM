LineChart

LineChart.init()
Initializes all the line charts
Here initializing means adding a physical object away from the visible area of the map. This is to ensure all the joints are working properly. NOTE: You must call this method in each new game
LineChart.handleClick(id, n, call)
idinteger
The ID of the data point (text area)
nstring
The name of the player
callstring
Event callback
This method handles click events to display data labels
There is no need to manually call this method inside your code. You need to just insert this inside the eventTextAreaCallback method Example 

            ...

            function eventTextAreaCallback(id, n, call)
                LineChart.handleclicks(id, n, call)
            end

            ...
This method usually triggers when a player click a data label
